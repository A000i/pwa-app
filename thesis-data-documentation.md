# 卒論用：実装済み公開データ学習システム詳細

## 1. NBA Official Statistics API

### データソース詳細

- **提供元**: NBA Entertainment (National Basketball Association)
- **公式 URL**: https://stats.nba.com/stats/
- **API エンドポイント**: `leaguedashplayerstats`
- **データ範囲**: 2023-24 シーズン レギュラーシーズン
- **対象選手数**: 約 400 名の現役 NBA 選手
- **アクセス方法**: 公開 REST API（利用制限あり）

### 取得データ項目

```
PLAYER_NAME: 選手名
TEAM_ABBREVIATION: 所属チーム
GP: 出場試合数
MIN: 平均出場時間
FGM: フィールドゴール成功数
FGA: フィールドゴール試投数
FG_PCT: フィールドゴール成功率
FG3M: 3ポイント成功数
FG3A: 3ポイント試投数
FG3_PCT: 3ポイント成功率
FTM: フリースロー成功数
FTA: フリースロー試投数
FT_PCT: フリースロー成功率
```

### 学術的価値

- 世界最高峰プロリーグの客観的統計データ
- 大規模サンプル（n=400 名以上）による統計的信頼性
- 標準化された計測・記録システム
- 研究・教育目的での利用が認められている

### 実装での活用方法

- 上位パーセンタイル（P90, P75, P50, P25）による評価基準設定
- プロレベルのベンチマーク値として機械学習の教師データに使用
- フィールドゴール成功率を基準とした 5 段階評価システム構築

## 2. Basketball-Reference Historical Data

### データソース詳細

- **提供元**: Sports Reference LLC
- **公式 URL**: https://www.basketball-reference.com/
- **データ範囲**: 2022 年〜2024 年（過去 3 シーズン）
- **対象**: NBA 全選手の詳細統計
- **データ量**: 約 1,200 名分の選手統計
- **アクセス方法**: 公開ウェブデータ（利用規約準拠）

### 取得統計項目

```
Season: シーズン年
Player: 選手名
Pos: ポジション
Age: 年齢
Tm: チーム
G: 出場試合数
GS: 先発出場数
MP: 出場時間
FG: フィールドゴール成功数
FGA: フィールドゴール試投数
FG%: フィールドゴール成功率
3P: 3ポイント成功数
3PA: 3ポイント試投数
3P%: 3ポイント成功率
2P: 2ポイント成功数
2PA: 2ポイント試投数
2P%: 2ポイント成功率
eFG%: 実効フィールドゴール成功率
FT: フリースロー成功数
FTA: フリースロー試投数
FT%: フリースロー成功率
```

### 学術的価値

- 長期間の統計トレンド分析が可能
- 複数シーズンにわたる一貫性検証
- アドバンスド・メトリクス（eFG%等）による詳細分析
- スポーツ統計学の標準的データソースとして学術界で認知

### 実装での活用方法

- 時系列データによる評価基準の安定性検証
- 過去 3 年間の統計分布から正規分布パラメータ推定
- 選手レベル別（エリート/平均/ルーキー）の基準値設定

## 実装システムでの統合利用

### データ統合手法

1. **統計的正規化**: 各データソースの尺度を 0-1 に正規化
2. **重み付け統合**: NBA 現役(1.0) > 学術研究(0.9) > 過去データ(0.8)
3. **パーセンタイル分析**: 統合データから評価閾値を自動計算
4. **クロスバリデーション**: 複数ソースでの一貫性検証

### 評価基準生成

```javascript
// 統計的閾値計算結果例
excellent: {
  fg_pct: 0.487,    // NBA上位10%パーセンタイル
  fg3_pct: 0.383,   // 3ポイント上位10%
  ft_pct: 0.891,    // フリースロー上位10%
  description: 'NBA上位10%レベル（エリート）'
},
good: {
  fg_pct: 0.452,    // NBA上位25%パーセンタイル
  fg3_pct: 0.351,   // 3ポイント上位25%
  ft_pct: 0.842,    // フリースロー上位25%
  description: 'NBA上位25%レベル（上級）'
},
average: {
  fg_pct: 0.421,    // NBA中央値（50%パーセンタイル）
  fg3_pct: 0.318,   // 3ポイント中央値
  ft_pct: 0.781,    // フリースロー中央値
  description: 'NBA平均レベル（中級）'
}
```
