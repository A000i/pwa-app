# バスケットボールシュートフォーム AI 学習計画

## 📊 利用可能データセット

### 1. Qlean Dataset（Visual Bank 社）

- **内容**: 3D 高精度バスケットボール動作データ
- **特徴**: プロ選手の標準シュートフォーム
- **形式**: 3D 座標データ + 動画
- **用途**: 基準モデル学習

### 2. 学術論文データ

- **近畿大学研究**: 関節角度データ比較
- **LSTM 研究**: フォーム解析モデル
- **J-STAGE 論文**: 自己学習システム設計
- **用途**: 評価アルゴリズム設計

### 3. データスタジアム

- **内容**: プロ・アマ選手試合データ
- **特徴**: 実戦での動作パターン
- **用途**: バリエーション学習

## 🤖 AI 学習アーキテクチャ

### Phase 1: 基準動作モデル構築

```
Qlean Dataset → 3D座標正規化 → 基準シュートフォーム抽出
                                ↓
                        TensorFlow.js学習モデル
```

### Phase 2: 評価システム統合

```
ユーザー動画 → PoseDetection → 座標抽出 → AI比較評価 → スコア算出
                                ↓
                        基準動作との差分解析
```

### Phase 3: 継続学習システム

```
新規データ → 評価フィードバック → モデル更新 → 精度向上
```
